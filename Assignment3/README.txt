1) The project contains one header file Quad.h

2) There is one implementation file Quad.cpp

3) There is one driver file QuadTestDriver.cpp which tests all the functionalities of Quad class. It also contains main function and is the starting point of project.

4) The main function of QuadTestDriver tests class Quad, specifically, tests constructor, compound assignment operator overload, pre/post increment/decrement, subscripts,
    function objects, input/output operators, equality relational operators, etc.

5) Compiled and executed using Visual Studio.
