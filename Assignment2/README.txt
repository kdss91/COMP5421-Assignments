1) The project contains two header files Command.h and LineEd.h

2) There are two implementation files Command.cpp and LineEd.cpp

3) There is one driver file LineEdDriver.cpp which tests all the functionalities of LineEd class. It also contains main function and is the starting point of project.

4) The run function of LineEd class takes command-line as an input from keyboard and executes each command one by one.

5) The testLineEd function of LineEdDriver tests various commands by calling runCommand function of LineEd class.

6) Compiled and executed using Visual Studio.
